## 比赛简介
Data Joy员工满意度预测赛，是科赛网(https://www.kesci.com)举办的一场数据科学竞赛。

## 赛题&数据
   **员工满意度**又称**雇员满意度**，是团队精神的一种参考，也就是个体对他所从事的工作的一般态度。是组织成员对其工作特征的认知评价，是员工通过比较实际获得的价值与期望获得的价值之间的差距后，对工作各方面满足与否的态度和情绪反映。
    通过持续的员工满意度调查，企业可以对自身管理中所存在的问题进行“诊断”，进而系统地解决问题，改善企业的管理，提高生产效率、降低人员流失率。因此，科学的员工满意度分析对于提高企业管理的有效性意义重大。
本次练习赛基于员工的满意度调查与员工的基本信息和工作经历，预测员工对于公司的**满意度情况**。
 

`数据说明`
    本次练习赛所使用数据集基于人力资源分析数据集，并且针对部分字段做出了一定的调整，所有的字段信息请以本练习赛提供的字段信息为准，字段信息内容参考如下：
|字段名 |数据类型|字段描述|
|------|-------|-------|
|id    |int    |数据的id|
|satisfaction_level|float|员工满意度，0-1分，待预测的值|
|last_evaluation|float|员工上一次内评获得的评价|
|average_monthly_hours|float|每月平均工作时长|
|time_spend_company|int|工龄(年)|
|number_project|int|在职期间完成的项目数量|
|work_accident|int|是否有工伤：0-没有，1-有|
|promotion_last_5years|int|过去5年是否有升职：0-没有，1-有|
|division|string|工作部门|
|salary|string|工资相对等级|
|package|string|福利待遇等级|
